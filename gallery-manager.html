🖼 Gallery Manager

SP Driving School · Add photos & videos visible to everyone
⚙️ One-Time Setup Do this once
Create a Google Sheet — this is your gallery database. Every photo/video URL you add here will appear on the website for everyone.
1
Go to sheets.google.com → click + New spreadsheet
2
In Row 1, add these exact column headers (copy-paste them):
urltypecategorycaption
3
Click File → Share → Publish to web → choose Sheet1 → format Comma-separated values (.csv) → click Publish → copy the URL it gives you
4
Open gallery.html in GitHub → find this line near the top:
Line to find:const SHEET_CSV_URL = 'YOUR_SHEET_CSV_URL_HERE';
Replace YOUR_SHEET_CSV_URL_HERE with the URL you copied → commit ✅
📸 How to Add a Photo Every time
Upload photo to Cloudinary → copy URL → paste in your Google Sheet. Done. Visible on all devices immediately.
1
Go to cloudinary.com → sign up free → click Media Library → Upload → pick your photo
2
After upload → click the photo → click Copy URL (ends in .jpg or .png)
3
Open your Google Sheet → add a new row:
url
Paste the Cloudinary URL
type
photo
category
e.g. prac-2w
caption
e.g. Sangita Ma'am class
4
Refresh the gallery page → photo appears for everyone on every device ✅
🎬 How to Add a Video Every time
Two options — YouTube (opens on YouTube) or Cloudinary (plays directly on your site with full controls).
Option A — YouTube (easiest): Upload to YouTube → set Public/Unlisted → copy the youtube.com/watch?v=… link → paste as URL in sheet → type = video
1
Option B — Cloudinary (plays on site): Go to cloudinary.com → upload video → copy URL ending in .mp4
2
Add a row in your Google Sheet:
url
YouTube or Cloudinary URL
type
video
category
e.g. review-video
caption
Customer name
3
Refresh gallery page → video appears ✅ Cloudinary videos play with full controls. YouTube videos open on YouTube when clicked.

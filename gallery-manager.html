🖼 Gallery Manager

SP Driving School · Manage photos & videos via Google Sheets

⚙️ One-Time Setup

Do once only
1
Go to sheets.google.com → click + Blank spreadsheet
2
In Row 1 add these exact column headers (copy-paste exactly):
urltypecategorycaption
3
Click File → Share → Publish to web → select Sheet1 → format Comma-separated values (.csv) → click Publish → copy the URL
4
Open gallery.html on GitHub → find this line:
const SHEET_CSV_URL = 'YOUR_SHEET_CSV_URL_HERE';
Replace YOUR_SHEET_CSV_URL_HERE with your copied URL → commit ✅

📸 Add a Photo

Every time
1
Go to cloudinary.com → sign up free → Media Library → Upload → pick your photo
2
After upload → click the photo → click Copy URL (ends in .jpg or .png)
3
Open your Google Sheet → add a new row like this:
url
https://res.cloudinary.com/…/photo.jpg
type
photo
category
prac-2w
caption
Sangita Ma'am class
4
Wait 1–2 minutes → refresh gallery page → photo appears for everyone on every device ✅

🎬 Add a Video

Every time
Option A — YouTube (video opens on YouTube)
1
Upload your video to youtube.com → set visibility to Public or Unlisted
2
Copy the YouTube URL → add row in Google Sheet:
url
https://youtube.com/watch?v=xxxx
type
video
category
review-video
caption
Rahul's Review
Option B — Cloudinary (plays directly on your site with controls)
1
Go to cloudinary.com → Media Library → Upload → pick your video
2
After upload → click the video → click Copy URL (ends in .mp4)
3
Add row in Google Sheet:
url
https://res.cloudinary.com/…/video.mp4
type
video
category
review-video
caption
Priya's Review
Both options appear in Customer Review Videos section.
Clicking a card opens a popup — YouTube plays in YouTube player · Cloudinary plays with full controls (play/pause/seek/mute/speed) · Use ‹ › arrows to go between videos
